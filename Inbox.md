## ✅ 処理チェックリスト（1件ずつ回す）

タスクを1件選んで以下を決める:

- [ ] 実行可能か？（No → [[Someday-Maybe]] か削除）
- [ ] 2分以内にできる？（Yes → 今すぐやって `x` をつける）
- [ ] 期日・開始日を設定した
- [ ] 見積時間を設定した（⏱15min / ⏱30min / ⏱1h / ⏱2h）
- [ ] プロジェクトまたはエリアに移動した
- [ ] コンテキストラベルを付けた
- [ ] 優先度を設定した（#P1〜#P4）
- [ ] Eisenhower象限を設定した（#Q1〜#Q4）

## 📌 タグ・記法リファレンス

### 優先度
| タグ | 意味 |
|------|------|
| `#P1` | 最重要（今日必ずやる） |
| `#P2` | 重要（今週中に） |
| `#P3` | 普通（余裕があれば） |
| `#P4` | 低（いつかでいい） |

### Eisenhower象限
| タグ | 意味 |
|------|------|
| `#Q1` | 緊急 × 重要 → 今すぐやる |
| `#Q2` | 余裕 × 重要 → 計画する |
| `#Q3` | 緊急 × 重要でない → 委任・最小化 |
| `#Q4` | 余裕 × 重要でない → 排除 |

### コンテキスト
| タグ | 意味 |
|------|------|
| `#context/home` | 家でできる |
| `#context/office` | オフィスが必要 |
| `#context/computer` | PCがあればできる |
| `#context/calls` | 電話・ミーティング |
| `#context/errands` | 外出が必要 |

### ステータス
| タグ | 意味 |
|------|------|
| `#next-action` | 今すぐ着手できる次のアクション |
| `#waiting` | 誰かの返事待ち |
| `#someday` | いつかやる |

### 日付・時間（Tasks plugin 対応）
```
📅 2025-04-10   ← 期日
🛫 2025-04-08   ← 開始日
⏱ 30min        ← 見積時間
🔁 every week   ← 繰り返し
```
